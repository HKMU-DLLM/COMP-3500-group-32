## Environment

Requires Node.js 20.x or later, but not 25.x

## How to install

In folder directory (e.g: `D:/COMP-3500-GROUP-32>`), run:

```
npm install
npm start
```

## How to use

Pre-setted Restaurant name:

-   McDonald's
-   Sushiro
-   Sharetea

If you messed up with the database, please copy the backup version in `/db_backup` to `/db`
